# Instagram Clone

This is a clone of Instagram. I have tried to achieve some functionality using JavaScipt, HTML and CSS.

## To run App -
run this command 

`npm run server`


## Pages & Features

- Social Sign in / Sign Up with Mock API
- Add Post
- Update Post
- Delete Post


## Tech Stack

- JavaScript 
- HTML
- CSS
